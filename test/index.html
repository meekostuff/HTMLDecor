HTMLDecor Tests

These are minimal visual tests which are hopefully self-explanatory.

When navigating away from a landing-page (and on browsers that support history.pushState()) the following line will be revealed.

pushState Assisted Navigation is active

The absence of this line indicates that the page is a landing-page - that is, it was NOT loaded with pushState.

Links to all these tests are included in the decor document and SHOULD appear at the top of each test-page.
